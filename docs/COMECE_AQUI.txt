🔧 PERSONALIZAÇÕES COMUNS:
════════════════════════════════════════════════════════════════════════════════

❓ Como adicionar novo produto?
   → Abra: components/products-carousel.tsx
   → Adicione novo objeto ao array 'products'
   → Salve imagem em: public/

❓ Como adicionar depoimento?
   → Abra: components/testimonials.tsx
   → Adicione novo objeto ao array 'testimonials'

❓ Como mudar cores?
   → Abra: app/globals.css
   → Edite as variáveis CSS em :root

❓ Como atualizar contato?
   → Abra: components/footer.tsx
   → Altere email, telefone, endereço

❓ Como fazer deploy?
   → Vercel (recomendado): vercel.com
   → Ou qualquer host Node.js


🚀 DEPLOY RÁPIDO (VERCEL):
════════════════════════════════════════════════════════════════════════════════

1. Faça push no GitHub
2. Vá em vercel.com
3. Clique "New Project"
4. Selecione seu repositório
5. Clique "Deploy"

Pronto! Seu site estará online em minutos!

🆘 PRECISA DE AJUDA?
════════════════════════════════════════════════════════════════════════════════

❌ Erro: "Cannot find module"
   → Rode: rm -rf node_modules .next && pnpm install

❌ Erro: "Imagens não carregam"
   → Verifique o caminho em /public/

❌ Erro: "WhatsApp não abre"
   → Formato correto: 55 + DDD + número (sem formatação)

❌ Erro: "Build falha"
   → Rode: pnpm tsc --noEmit (para ver o erro)
